# iplookup
查询ip或者域名信息

```
Usage: iplookup [OPTIONS]

Options:
  -a, --api <API>                    The API [baidu, ipinfo, taobao] to use
                                     Example: iplookup -a baidu -i 127.0.0.1 [default: baidu]
  -i, --ip <IP>                      The IP address to query, Example: iplookup -i 127.0.0.1
  -d, --domain <DOMAIN>              The domain to query, Example: iplookup -d example.com
  -I, --multi-ip <MULTI_IP>          The multi-ip address to query, Example: iplookup -I 127.0.0.1 -I 127.0.0.1
  -D, --multi-domain <MULTI_DOMAIN>  The multi-domain to query, Example: iplookup -D example.com -D example.com
  -h, --help                         Print help
  -V, --version                      Print version
```
